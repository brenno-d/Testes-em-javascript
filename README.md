1. Solicite ao usuário um número e informe se o mesmo é par ou ímpar; 
2. Solicite ao usuário um número e exiba sua tabuada de 1 a 10; 
3. Solicite a idade do usuário e informe se ele é criança (0 à 12), adolescente 
(13 à 17), adulto (18 à 59) ou idoso (60+); 
4. Solicite o peso e altura do usuário e calcule seu IMC; 
5. Desenvolva um sistema que converte temperaturas de Celsius para 
Fahrenheit e Kelvin; 
6. Desenvolva um sistema que exiba todos os números pares entre 0 e 100; 
7. Desenvolva um sistema que realize a soma de todos os números digitados 
pelo usuário até ele digitar o número 0. Após isso, exiba o resultado da soma; 
8. Desenvolva um sistema que solicite ao usuário a sua data de nascimento, 
depois exiba sua idade.
